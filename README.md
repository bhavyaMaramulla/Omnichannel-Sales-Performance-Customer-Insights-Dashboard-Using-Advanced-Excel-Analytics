# Omnichannel-Sales-Performance-Customer-Insights-Dashboard-Using-Advanced-Excel-Analytics
Leverage multi-channel sales analytics to uncover performance patterns across Website, App, In-Store, and Phone channels. Apply structured analysis to derive product and channel trends.

## Project Overview
This project focuses on analyzing multi-channel sales data to uncover performance trends, customer behavior patterns, and product-level insights using Advanced Microsoft Excel.
The dashboard consolidates transactional data from multiple sales channels including Website, Mobile App, In-Store, and Phone orders. The objective was to transform raw structured data into meaningful, decision-oriented insights through systematic analysis and interactive reporting.
This project demonstrates my ability to approach business problems analytically and convert data into actionable intelligence.

# Project Objectives

Analyzed multi-channel sales data (Website, App, In-Store, Phone) to identify performance patterns
Evaluated product-level and channel-level trends using structured data analysis
segmented customers based on profession and purchasing behavior
designed an interactive dashboard for simplified decision-making and 
transformed raw transactional data into actionable business insights
